# [usecoherence.dev](https://usecoherence.dev)

Public website for Coherence — a bounded-claim machine for software engineering.

Built with [Eleventy](https://www.11ty.dev/). Deployed to [pgs.sh](https://pico.sh/pgs).

```text
src/      Markdown, templates, and assets
public/   generated output (gitignored)
```

## Commands

```bash
npm install
npm run dev      # local dev server
npm run build    # static build
npm run deploy   # build + rsync to pgs.sh
```
